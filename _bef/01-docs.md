## Summary

The Browser Extension Finder or BEF is a terminal application written in
Go that searches for installed plugins on Windows and Linux systems.
This tool mines user profiles on a system in search of any data it can
find in relation to browser plugins, extensions, etc.

The hope is to down the line turn this into an application that will
search through accounts recursively or a specific user profile on a
system. Though, we are very far from reaching that goal.

This project was inspired by the following project that was written in
Python.

- [brad_anton/extension_finder](https://github.com/brad-anton/extension_finder)
