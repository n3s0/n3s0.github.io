## Scenario
---

Scenario: "Cape Town": Borked Nginx

Level: Medium

Type: Fix

Access: Email

Description: 

There's an Nginx web server installed and managed by systemd. Running 
`curl -I 127.0.0.1:80` returns `curl: (7) Failed to connect to localhost port 80: Connection refused`, 
fix it so when you curl you get the default Nginx page.

Root (sudo) Access: True

Test: `curl -Is 127.0.0.1:80|head -1` returns `HTTP/1.1 200 OK`

Description=The NGINX HTTP and reverse proxy server

[Service]

ExecStart=/usr/sbin/nginx
ExecReload=/usr/sbin/nginx -s reload
