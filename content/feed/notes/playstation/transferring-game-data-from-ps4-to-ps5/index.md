## Summary

Although the PlayStation 5 has been out for a while. I'm usually pretty slow to
join the party on the latest and greatest consoles. This note discusses moving
some save data from the PlayStation 4 to the PlayStation 5.

## Data Transfer

There are a few methods to transferring data from PlayStation 4 to PlayStation
5.

1. Transfer using a USB device.
2. Transfer using the local network with a PlayStation Plus account.

> I'll get to both. But, for now. I'm going to post here the USB method.
> Pictures will be added to this note later.

### Transfer Using USB

> For those following along. Make sure to do this on the local profile where the
> saved data is located.

I will say that this method is the simplest. You can count on it working unless
the game data is corrupt or you're using NTFS or something. I prefer it because
it doesn't require you to sign in to PlayStation Plus.

Things you'll need are the following to go through this process.

- USB Flash Drive/Hard Drive (Formatted with exFAT)
- PlayStation 4
- PlayStation 5

Although some of these game saves can be just MBs. We'll want to setup the USB
drive to support larger data transfers. So, make sure to format it for ExFAT.
This can be done from either PlayStation.

First, connect the external storage to the PlayStation 4.

Second, go to "Copy to USB Storage Device".

```txt
Settings -> Application Saved Data Management -> Saved Data in System Storage ->
Copy to USB Storage Device
```

Then select which game save you'd like to save. This can either be all of the
game data on the profile. It can also be individual saves. Once finished with
selection. Select the `Copy` option and let it transfer.

Once this is complete. Disconnect the external drive from the PlayStation 4 and
connect it to the PlayStation 5.

Now copy the data to the PlayStation 5. This can be done by going to `Copy to
Console Storage` and selecting the saved data and transferring it. 

```txt
Settings -> Saved Data and Game/App Settings -> Saved Data (PS4) -> USB Drive
```

Pick what game data will be transferred and move them over.

> Make sur to open the game and load the game save to confirm that it works.
