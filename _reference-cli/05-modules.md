## Overview

Provided is a list of modules providing their functionality and
describes what they're for. I've added a little preview to each 
module so users have an idea of what each looks like.

## Modules

### NATO Phonetic Alphabet (natopa)

This module outputs a table of NATO's Phonetic Alphabet.

```sh
++++++++++ Phonetic Alphabet ++++++++++++

* Credit for Table goes to http://www.lookuptables.com/

++++++++ NATO Phonetic Alphabet +++++++++
+--------+--------------+---------------+
| Letter | Written Word | How To Say It |
+--------+--------------+---------------+
|   A    |    Alpha     |     Al-fah    |
|   B    |    Bravo     |    Brah-voh   |
|   C    |   Charlie    |    Char-lee   |
|   D    |    Delta     |    Dell-tah   |
|   E    |     Echo     |     Eck-oh    |
|   F    |   Foxtrot    |   Foks-trot   |
|   G    |     Golf     |      Golf     |
|   H    |    Hotel     |    Hoh-tel    |
|   I    |    India     |   In-dee-ah   |
|   J    |    Juliet    |   Jew-lee-et  |
|   K    |     Kilo     |    Key-loh    |
|   L    |     Lima     |    Lee-mah    |
```

### International Morse Code (int-morse)

Outputs a table of the International Morse Code patterns and what they
mean.

```sh
Morse Code Letters
+------------------+----------------+------------------+----------------+
| Letters - Part 1 | Morse - Part 1 | Letters - Part 2 | Morse - Part 2 |
+------------------+----------------+------------------+----------------+
|        A         |       .-       |        N         |       -.       |
|        B         |      -...      |        O         |      ---       |
|        C         |      -.-.      |        P         |      .--.      |
|        D         |      -..       |        Q         |      --.-      |
|        E         |       .        |        R         |      .-.       |
|        F         |      ..-.      |        S         |      ...       |
|        G         |      --.       |        T         |       -        |
|        H         |      ....      |        U         |      ..-       |
|        I         |       ..       |        V         |      ...-      |
|        J         |      .---      |        W         |      .--       |
|        K         |      -.-       |        X         |      -..-      |
|        L         |      .-..      |        Y         |      -.--      |
|        M         |       --       |        Z         |      --..      |
+------------------+----------------+------------------+----------------+

Morse Code Digits
+-------+-------+
| Digit | Morse |
+-------+-------+
|   0   | ----- |
|   1   | .---- |
|   2   | ..--- |
|   3   | ...-- |
|   4   | ....- |
```
