## Summary

Don't quite know what to name this yet. Just going with something
random.

This is the initial page for the DiffInv documentation. This is an
inventory management application that includes other useful features.
I hope to design this application to do the following. This will be
mostly catered to make life easier for IT professionals.

There are a lot of applications out there. But, a lot of them only cater
to only one aspect of Asset Management, Datacenter & Infrastructure
Management, and the like. I would like to make an application that takes
all of the good things from applications like Snipe-IT, netbox, dashy,
etc and put them all into one application.

This road is going to be a long one because I've never developed an
application for such a thing. But, if I can execute on the vision of
this project. It will be truely extrordinary. I will have everything I
have ever wanted in an application for my datacenters, infs, and for the
assets I have managed over the years.

This will require a LOT of planning and preparation before I even start
coding for it.
