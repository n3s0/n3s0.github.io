## Overview

This is the script I'm using as proof of concept code for solutions 
involving challenges from The Python Challenge. Should there be any 
issues with the code, please submit an issue.

This code repository is used to compliment the following post on my 
blog.

- [The Python Challenge - Writeup](https://code.n3s0.tech/2021/12/18/the-python-challenge)

## Clone This Repository

If you would like to look at the code on your local machine. You're 
welcome to clone it. Below is how one can clone it should you want to.

```sh
git clone https://github.com/n3s0/python-challenge-s.git
```

## Running

Script provides a usage and a help. You can find this by running the 
following command.

```sh
python python-challenge.py
```

Or by running the following command.

```sh
python python-challenge.py -h
```

Below is the output from the help/usage.

```sh
python python-challenge.py
usage: Python Challenge Solutions [-h] [-l] [-s SOLUTION]

Solutions for The Python Challenge.

optional arguments:
  -h, --help            show this help message and exit
  -l, --list            List the available solutions
  -s SOLUTION, --solution SOLUTION
                        Selected option from solution

Written by Timothy Loftus (n3s0)
```
