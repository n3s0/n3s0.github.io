## Summary

This is the initial page for the IR Remote Documentation. Anything
related to IR Remote can be found here. Anyone can find documentation on
the assembly of the parts, usable parts, documentation on the software,
how to configure each piece, etc. 

## Project Overview

This project started with a little problem where I didn't have a remote
one day to my Vizio TV and didn't want to spend the change on a 
universal remote. So I wrote an application and put together the parts
for an Arduino so I could at least turn it on.
